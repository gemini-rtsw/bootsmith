Repair Board Information Block (VPD) on {{ profile.name }}

One-shot operation. This writes the board's VPD EEPROM (the data shown by CNFG). Only run this when CNFG on the board prints "Board Information Block Checksum Error" or shows garbage like UUUUUU / ?????? — usually after a dead backup battery.

Values are per-board and not saved in the profile. Most fields you can fake; the only one that matters is the Ethernet Address (MAC), which must be unique and match the sticker on the board.
